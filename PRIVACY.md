# プライバシーポリシー

最終更新日: 2025年1月16日

## はじめに

Chrome to Medical（以下「本拡張機能」）は、医療従事者の診療記録作成業務を支援するChrome拡張機能です。本ポリシーでは、本拡張機能がどのようにデータを取り扱うかを説明します。

## 開発者情報

- 拡張機能名: Chrome to Medical
- バージョン: 0.1.0
- リポジトリ: https://github.com/NikoToRA/Chrome_to_Medical

## データ収集と使用

### 収集しないデータ

本拡張機能は以下のデータを**収集しません**：

- ✅ 個人を特定できる情報（氏名、住所、電話番号等）
- ✅ 患者情報
- ✅ 診療記録
- ✅ ブラウジング履歴
- ✅ 使用統計・分析データ
- ✅ クッキーやトラッキングデータ

### ローカルに保存されるデータ

以下のデータは**お使いのデバイスのみ**に保存され、外部に送信されません：

#### 1. テンプレートデータ
- 内容: ユーザーが作成した固定文テンプレート
- 保存場所: Chrome Storage API（ローカルストレージ）
- 保存期間: 拡張機能がインストールされている間
- 削除方法: 拡張機能のアンインストール、または設定画面から手動削除

#### 2. ユーザー設定
- 内容: 表示設定、カテゴリ設定等
- 保存場所: Chrome Storage API（ローカルストレージ）
- 保存期間: 拡張機能がインストールされている間

#### 3. APIキー（オプション）
- 内容: Claude API のAPIキー（AI機能を使用する場合のみ）
- 保存場所: Chrome Storage API（ローカルストレージ）
- 保存期間: ユーザーが削除するまで
- 削除方法: 設定画面から削除可能

### 外部に送信されるデータ

#### Claude API（Anthropic社）への送信

**AI診療記録作成機能を使用する場合のみ**、以下のデータが送信されます：

- 送信されるデータ: ユーザーが入力したプロンプトテキスト
- 送信先: Claude API (https://www.anthropic.com/)
- 目的: AI による診療記録の生成
- 同意: 機能使用前に毎回明示的に同意を取得
- 確認: 送信されるデータは事前に確認可能

**重要**: 個人を特定できる情報や機密性の高い患者情報を含めないことを強く推奨します。

**Anthropic社のプライバシーポリシー**: https://www.anthropic.com/privacy

## データの保持と削除

### ローカルデータの保持期間
- テンプレートデータ: 拡張機能がインストールされている間
- ユーザー設定: 拡張機能がインストールされている間

### データの削除方法

#### 全データの削除
1. Chrome拡張機能管理ページ（`chrome://extensions/`）を開く
2. Chrome to Medical の「削除」ボタンをクリック
3. 全てのローカルデータが完全に削除されます

#### 個別データの削除
- テンプレート: 設定画面から個別に削除可能
- APIキー: 設定画面から削除可能

### 外部サービスでのデータ保持
- **Claude API**: Anthropic社のデータ保持ポリシーに準拠します
  - 詳細: https://www.anthropic.com/privacy

## 第三者サービスの利用

本拡張機能は以下の第三者サービスを利用します：

### 1. Claude API (Anthropic社)
- 用途: AI診療記録作成機能
- 使用タイミング: ユーザーが明示的にAI機能を使用した場合のみ
- データの流れ: ブラウザ → Claude API → ブラウザ
- プライバシーポリシー: https://www.anthropic.com/privacy

本拡張機能の開発者は、Anthropic社のデータ取り扱いについて責任を負いません。

## セキュリティ

### データ保護措置
- 全ての外部通信はHTTPS経由で暗号化
- ローカルストレージは Chrome のセキュリティ機能で保護
- Manifest V3 準拠（最新のセキュリティ基準）
- Content Security Policy (CSP) による保護

### 計画中のセキュリティ強化
- APIキーの暗号化保存（Phase 2）
- テンプレートデータの暗号化（Phase 2）
- エクスポート機能の暗号化（Phase 2）

## 権限の使用目的

本拡張機能が要求する権限と使用目的：

| 権限 | 使用目的 |
|------|---------|
| `sidePanel` | サイドパネルUIの表示 |
| `storage` | テンプレートと設定のローカル保存 |
| `tabs` | 現在のタブ情報の取得（プラットフォーム検出用） |
| `clipboardWrite` | 生成したテキストのクリップボードへのコピー |
| `scripting` | 電子カルテシステムへのテキスト挿入 |
| `offscreen` | クリップボード操作の補助処理 |
| `<all_urls>` | 任意の電子カルテシステムでの動作 |

## お子様のプライバシー

本拡張機能は13歳未満のお子様を対象としていません。13歳未満のお子様からデータを意図的に収集することはありません。

## プライバシーポリシーの変更

本ポリシーは予告なく変更される場合があります。重要な変更がある場合は、拡張機能内で通知します。

変更履歴はGitHubリポジトリで確認できます：
https://github.com/NikoToRA/Chrome_to_Medical/commits/main/PRIVACY.md

## 推奨事項

医療情報の性質を考慮し、以下を推奨します：

### ✅ 推奨される使用方法
1. 患者情報は匿名化して使用
2. AI機能使用時は機密性の低い情報のみ送信
3. 重要な記録は必ず電子カルテシステムに保存
4. 共有端末では使用後にログアウト

### ❌ 推奨されない使用方法
1. 実名や詳細な個人情報の入力
2. 未確認のAI生成結果をそのまま使用
3. 本拡張機能のみでの記録保管
4. 組織のポリシーに反する使用

## 免責事項

本拡張機能は診療業務の補助ツールです。以下の点にご注意ください：

- 医療判断の代替となるものではありません
- 生成された記録は必ず医療従事者が確認・修正してください
- 正式な診療記録は認可された電子カルテシステムに記録してください
- 本拡張機能の使用により生じた損害について、開発者は責任を負いかねます
