# 第1話 3ページ目：転（疑念と真実）

### コマ1：疑念（ベンダー調整）
- **絵の構成**: 医師、呆れた顔で手を振る（やれやれという感じ）。
- **セリフ（医師）**: 「真っ当な医療ツール？導入するのに電子カルテメーカーとの調整とか、院内LAN工事とか大変なんだよ。そんな簡単に…」
- **セリフ（エージェント）**: 「それ、全部不要です！」

### コマ2：反論（完全不要）
- **絵の構成**: エージェントがバツ（×）のポーズ、あるいは既存の重厚なサーバー機器を蹴飛ばすイメージ。
- **セリフ（エージェント）**: 「ベンダー調整ゼロ！工事ゼロ！既存のPCにChrome拡張機能を入れるだけ！」
- **セリフ（医師）**: 「はあ？そんなのアリかよ…」

### コマ3：疑念（コスト）
- **絵の構成**: 医師、鋭い目つきで問い詰める。
- **セリフ（医師）**: 「…だとしても高いんだろ？初期費用◯百万、月額◯万とか…」
- **セリフ（医師）**: 「うちは開業したてで金ないんだよ。無料なんてどうせ嘘だろ」

### コマ4：真実（圧倒的安さと手軽さ）
- **絵の構成**: エージェントが電卓や価格表（0円）を見せる。
- **セリフ（エージェント）**: 「嘘じゃありません！初期費用0円、月額もランチ代程度です！」
- **セリフ（エージェント）**: 「まずは2週間の完全無料トライアルで、その『疑い』を晴らしてください！」
- **医師リアクション**: 「（ランチ代…だと…！？）」
