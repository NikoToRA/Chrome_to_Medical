# 漫画制作ディレクトリ運用ルール

このディレクトリは、漫画制作プロジェクトの各アセットを管理するために使用します。
エージェントと共に以下のルールに従ってファイルを整理してください。

## ディレクトリ構成と役割

### 1. `story/` (ストーリー・脚本)
- **格納するもの**: プロット、あらすじ、脚本（シナリオ）、セリフノートなど。
- **命名規則**: `chapter1_plot.md`, `customer_journey.md` など。

### 2. `charactor/` (キャラクター設定)
- **格納するもの**: キャラクタープロフィール、性格設定、外見の特徴、参考画像、画像生成プロンプトなど。
- **ファイル例**: `profiles.md`, `agent_lineart.png`, `doctor_flat_color.png`。

### 3. `comic/` (コミック・原稿)
- **格納するもの**: ネーム（絵コンテ）、下書き、ペン入れ済み原稿、完成原稿など。
- **命名規則**: `page01_name.jpg`, `chapter1_finished.pdf` など。

### 4. `log/` (制作ログ)
- **格納するもの**: 制作日誌、打ち合わせメモ、アイデアの断片、ToDoリストなど。
- **運用**: エージェントとの対話ログや、その日の進捗を記録するのに使用する。

## エージェントの活用
「漫画制作管理エージェント」は、このルールに従ってファイルの整理をサポートします。
ファイルの保存場所に迷った時や、新しいアイデアをどこに記録すべきか相談したい時は、エージェントに聞いてください。
