Karte AI+ プライバシーポリシー
================================

本プライバシーポリシーは、Wonder Drill株式会社（以下「当社」といいます）が提供する Chrome 拡張機能「Karte AI+」（以下「本拡張機能」といいます）が取り扱うユーザー情報の取扱いについて定めるものです。本拡張機能をご利用いただくにあたり、本ポリシーの内容をご確認ください。


## 1. 収集する情報

本拡張機能は、以下の情報を取り扱うことがあります。

- **拡張機能内でユーザーが入力または送信する情報**
  - サイドパネルに入力されたテキスト（診療記録の下書き、メモ、指示文など）
  - ユーザーが明示的に拡張機能から送信操作を行ったページ内容（選択したテキスト、スクリーンショット等）
- **認証およびアカウント関連情報**
  - 認証トークン
  - メールアドレス
  - セッション ID 等の識別子
- **拡張機能の設定情報**
  - AI エージェントの設定（名前、説明、システムプロンプト等）
  - 選択中の AI モデル
  - ユーザー自身が入力した API キー
- **決済および領収書に関する情報**
  - 決済に利用された名前、メールアドレス、支払金額などの請求情報
  - 決済プロバイダ（例：Stripe 等）が発行する領収書・請求書に必要な情報

本拡張機能は、ユーザーが明示的に操作した範囲を除き、ブラウジング履歴全体や全ページの内容を継続的に収集することはありません。


## 2. 情報の利用目的

本拡張機能は、取得した情報を以下の目的に限定して利用します。

- クラウド型電子カルテ等における**診療記録作成支援機能**の提供
- テンプレート文や定型文の挿入、テキスト整形等のサポート機能の提供
- ユーザー認証およびログイン状態の維持
- 拡張機能の設定内容の保存・復元
- 利用料金の請求、決済処理、領収書・請求書の発行およびそれに付随する会計・経理処理
- サービス品質の向上・不具合対応のための分析（個人を特定しない統計的な情報に限る場合）

これらの目的以外で情報を利用することはありません。


## 3. 外部サービス・第三者提供について

本拡張機能は、以下のような外部サービスとの連携のために、ユーザーの情報を送信することがあります。

- **当社バックエンド（Microsoft Azure）への送信**  
  認証、ライセンス管理、AI 生成処理等を行うため、認証トークン、メールアドレス、セッション ID、利用ログ等を当社が管理する Microsoft Azure 上のサーバーへ送信し、安全に保管・処理します。

- **決済プロバイダへの送信**  
  利用料金の決済処理および領収書の発行等のため、氏名、メールアドレス、支払金額等の請求情報を決済プロバイダ（例：Stripe 等）に送信することがあります。これらの情報は、決済処理および会計上必要な範囲に限定して利用されます。

これらの場合であっても、送信される情報は診療記録作成支援および関連サービスの提供に必要な範囲に限定し、広告目的での利用やデータ販売は行いません。  
当社は、厚生労働省・総務省・経済産業省が策定する「いわゆる 3 省 2 ガイドライン」（医療情報システムの安全管理に関するガイドライン等）に準拠した体制で情報を取り扱います。


## 4. データの保存方法・保存期間

- ブラウザ内に保存される情報（設定情報、認証トークン等）は、`chrome.storage.local` などの仕組みを用いて、ユーザーの端末上に保存されます。
- これらの情報は、本拡張機能の機能提供に必要な期間に限り保持されます。
- ユーザーは、拡張機能の削除やブラウザのサイトデータ削除等により、ブラウザ内に保存されたデータを削除できます。

決済・請求情報やログ等、サーバー側に保存される情報については、サービス運営および会計・税務上必要な期間保持し、その後は適切な方法で削除または匿名化します。これらの保管・運用も、前記 3 省 2 ガイドラインに準拠したプロセスに従います。


## 5. セキュリティ

当社は、ユーザー情報への不正アクセス、紛失、破壊、改ざん及び漏えい等を防止するために、Microsoft Azure のセキュリティ機能を活用しつつ、必要かつ適切な技術的・組織的安全管理措置を講じます。  
また、外部サービスとの通信には HTTPS 等の暗号化された通信手段を用いるよう努めます。


## 6. ユーザーの選択とコントロール

- ユーザーは、拡張機能を利用しないことを選択することができます。
- ブラウザの機能を用いて、いつでも本拡張機能を無効化または削除することができます。
- クリップボードやページ内容へのアクセスは、ユーザーが拡張機能を操作したときに限って実行され、バックグラウンドで無断に行うことはありません。


## 7. 法令遵守

当社は、適用される日本の個人情報保護関連法令およびその他の関係法令に加え、厚生労働省・総務省・経済産業省による医療情報システム関連ガイドライン（3 省 2 ガイドライン）を遵守し、ユーザー情報を適切に取り扱います。
