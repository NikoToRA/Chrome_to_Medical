# 操作フロー（MVP）

## 基本フロー：ホットキー→AI→自動貼り付け

1. ユーザーはMAPSの入力欄にカーソルを置く
2. ユーザーがホットキーを押す（例：Alt+Space）
3. 画面上に小型AIパネルが表示される
4. ユーザーがテンプレ（SOAP/要約など）を選択し、必要ならメモを追記
5. 実行ボタンを押す
6. 生成中は進捗（スピナー/残り）を表示
7. 生成完了
8. アプリがMAPSへフォーカスを戻す
9. 生成結果をクリップボードへ格納
10. Ctrl+V を自動送出して貼り付け
11. 貼り付け完了をトースト通知

## 例外フロー：貼り付け失敗

- フォーカス復帰に失敗 → 2回までリトライ → 失敗通知（貼り付け未完了）
- Ctrl+Vが効かない → UIAでの直接入力を試行（将来）/ 失敗通知
