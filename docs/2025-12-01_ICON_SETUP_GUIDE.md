# アイコン画像の設定ガイド
作成日: 2025-12-01

## 📋 アイコン画像の要件

Chrome Web Storeに公開するには、以下のサイズのアイコン画像が必要です：

### 必須
- **128x128 PNG**: Chrome Web Storeの表示用（必須）

### 推奨
- **16x16 PNG**: ブラウザのツールバー表示用
- **48x48 PNG**: 拡張機能管理画面表示用

## 📁 ファイル配置

アイコン画像は以下の場所に配置してください：

```
extensions/icons/
  ├── icon-16.png  (16x16 ピクセル)
  ├── icon-48.png  (48x48 ピクセル)
  └── icon-128.png (128x128 ピクセル) ← 必須
```

## ✅ 設定状況

現在、`manifest.json` には以下の設定が追加されています：

```json
"action": {
  "default_title": "Chrome to Medical を開く",
  "default_icon": {
    "16": "icons/icon-16.png",
    "48": "icons/icon-48.png",
    "128": "icons/icon-128.png"
  }
},
"icons": {
  "16": "icons/icon-16.png",
  "48": "icons/icon-48.png",
  "128": "icons/icon-128.png"
}
```

## 🔧 次のステップ

1. **アイコン画像を配置**
   - `extensions/icons/` ディレクトリに画像ファイルを配置
   - ファイル名は `icon-16.png`, `icon-48.png`, `icon-128.png` にしてください

2. **画像サイズの確認**
   - 各画像が正しいサイズ（16x16, 48x48, 128x128）であることを確認
   - PNG形式であることを確認

3. **動作確認**
   - 拡張機能をリロード
   - ブラウザのツールバーにアイコンが表示されることを確認
   - 拡張機能管理画面でアイコンが表示されることを確認

## 📝 注意事項

- アイコン画像が見つからない場合、Chromeはデフォルトアイコンを表示します
- 128x128のアイコンは必須です。これがないとChrome Web Storeでの審査が通らない可能性があります
- アイコンは正方形である必要があります（16x16, 48x48, 128x128）
- PNG形式を推奨します（透明背景も可能）

## 🎨 アイコンデザインの推奨事項

- シンプルで分かりやすいデザイン
- 小さなサイズでも認識しやすい
- ブランドカラーを使用（Karte AI+のブランドカラー）
- 医療関連のアイコン（例: カルテ、医療記録など）
