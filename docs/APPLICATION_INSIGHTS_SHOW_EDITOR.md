# Application Insights - クエリエディタを表示する方法

**状況**: 結果テーブルが表示されていて、クエリエディタが見えない

---

## 🎯 クエリエディタを表示する方法

### 方法1: 画面上部の「traces」ボタンから開始（最も簡単）

1. **画面上部のツールバー**を見る
2. 「**traces**」という**青いボタン**が表示されているはず
3. その「**traces**」ボタンを**クリック**
4. クエリエディタが表示される
5. 自動的に `traces` というクエリが入力される
6. そのクエリを以下のように**編集**：

```
traces
| where timestamp > ago(1h)
| where message contains "AuthVerifyToken" or operation_Name contains "auth-verify-token"
| order by timestamp desc
| take 50
```

7. **実行**ボタンをクリック

---

### 方法2: 結果テーブルの上をクリック

1. **結果テーブルの上**（クエリタブの下あたり）を探す
2. クエリエディタが折りたたまれている可能性がある
3. 結果テーブルの**上端**を**ダブルクリック**または**上にドラッグ**
4. クエリエディタが表示される

---

### 方法3: 新しいクエリタブを開く

1. **画面上部のタブ**を見る（「New Query 1*」「New Query 2*」など）
2. **「+」ボタン**をクリック（新しいクエリタブを開く）
3. 新しいタブが開く
4. **左側のナビゲーションペイン**で「**traces**」を探す
5. 「**traces**」を**ダブルクリック**
6. クエリエディタに `traces` が表示される
7. 上記のクエリに編集

---

### 方法4: 左側のテーブルから直接選択

1. **左側のナビゲーションペイン**を確認
2. 「**Application Insights**」セクションを展開
3. 「**traces**」を探す
4. 「**traces**」を**ダブルクリック**
5. 新しいクエリタブが開き、クエリエディタが表示される
6. 上記のクエリに編集

---

## 📝 コピー用クエリ

以下のクエリをコピーして、クエリエディタに貼り付けてください：

```
traces
| where timestamp > ago(1h)
| where message contains "AuthVerifyToken" or operation_Name contains "auth-verify-token"
| order by timestamp desc
| take 50
```

---

## ✅ 確認ポイント

- クエリエディタは**結果テーブルの上**に表示されます
- 白い大きなテキストエリアがクエリエディタです
- クエリエディタが見えない場合は、結果テーブルを上にスクロールするか、折りたたみを解除してください
